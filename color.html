How it Works

As you know a color is made up of 3 seperate levels of red, blue, and green. My formula first substracts each from the other like so: lets say we're using the colors pink (255, 128, 192), and this shade of purple (71, 61, 139) Rnew = 255R - 71R
Gnew = 128G - 61G
Bnew = 192B - 139B
// This gives us a new 'rgb' value of (x, y, z)
Next you take our new rgb values and multiply them each by a bais between 0.0 and 1.0.
(Lets say the bias is 0.5)
Rnew = x * 0.5 Gnew = x * 0.5 Bnew = x * 0.5
After that you add the new values to the values of the first color.
Rnew Gnew Bnew
