Color Mixer

  This project was to demonstrate the different ways in which colors could be mixed. The earliest version was created as a way to rate movies. Users would rate a movie from 1 to 10. Each number corresponded to a color (0 being red, 10 being green, and 5 in the middle would be yellow.

 Athough it is apealing to see the change in color from one state to the next, the versatility of very limited.
The next step was to improve upon this system in order to mix any 2 colors together.
